Visualizations and statistics
based on the PDB data

  • X-ray data processing Interactive exploration of how detectors and the software in use differ between synchrotrons, and how they change in time. Plus other correlations.
  • Synchrotron work patterns What can we tell from data collection dates in the PDB?
  • PDBx/mmCIF tags A table that shows which tags are actually in use.
  • Frequency of chemical components A table that counts in how many entries each residue and ligand is present.
These statistics also show how to use project GEMMI (in particular one of the utilities: gemmi-grep) to quickly extract data from mmCIF files. For techinical details and more information see the source code repository.
